# 設定値

## Address

0〜0xFFFFの範囲。10進数で、0〜65535の範囲。

## Channel

Bandwidthによって、設定できるチャンネル値が変わります。

| Channel | Bandwidth(kHz) |
|:--|:--|
| 0-37  | 125kHz |
| 0-36  | 250kHz |
| 0-30  | 500kHz |
